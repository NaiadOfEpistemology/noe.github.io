JavaScript Events

What are JavaScript Events?

Events are actions or occurrences that happen in the browser. Examples include user interactions like clicking, hovering, or typing, and system events like page loading or resizing the window.

Adding Event Listeners

Event listeners allow JavaScript to "listen" for events on an element and execute a function when the event occurs. Use addEventListener() to register event handlers.


// Example of addEventListener
document.getElementById("myButton").addEventListener("click", function() {
    alert("Button clicked!");
});

Common Event Types

JavaScript supports various types of events, such as:

  • Mouse Events: click, dblclick, mouseover, mouseout
  • Keyboard Events: keydown, keyup, keypress
  • Form Events: submit, change, input
  • Window Events: load, resize, scroll

Inline Event Handling

Inline event handling involves adding event listeners directly within the HTML element using attributes like onclick, onchange, or onmouseover.

Removing Event Listeners

Use removeEventListener() to detach an event listener from an element.


// Example of removeEventListener
function sayHello() {
    alert("Hello!");
}
const button = document.getElementById("myButton");
button.addEventListener("click", sayHello);

// Later, remove the event listener
button.removeEventListener("click", sayHello);

Event Object

The event object provides information about the event and allows you to interact with it. It is automatically passed to event handler functions.


// Example of Event Object
document.getElementById("myButton").addEventListener("click", function(event) {
    console.log("Event Type: " + event.type);
    console.log("Target Element: " + event.target);
});
