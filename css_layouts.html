CSS Layouts

Box Model

The CSS Box Model describes the layout and design of elements. It includes margins, borders, padding, and the actual content.


/* Example CSS */
div {
    margin: 20px;
    padding: 15px;
    border: 1px solid black;
    width: 200px;
}

Positioning

CSS provides different positioning schemes for layout design:

  • Static: Default positioning.
  • Relative: Positioned relative to its normal position.
  • Absolute: Positioned relative to its nearest positioned ancestor.
  • Fixed: Positioned relative to the viewport.

/* Example of Positioning */
div {
    position: relative;
    top: 10px;
    left: 10px;
}

Flexbox

Flexbox is a powerful layout module used to design flexible and responsive layouts.


/* Example of Flexbox */
.container {
    display: flex;
    justify-content: space-between;
}
.child {
    flex: 1;
    margin: 5px;
}

Grid

The CSS Grid Layout is a two-dimensional layout system that allows you to design web pages easily.


/* Example of Grid Layout */
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
}
.grid-item {
    background: lightblue;
    padding: 20px;
    text-align: center;
}
