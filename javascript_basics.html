JavaScript Basics

Variables

JavaScript supports three types of variables:

  • var: Function-scoped, hoisted.
  • let: Block-scoped, preferred for most use cases.
  • const: Block-scoped, immutable (cannot be reassigned).

// Example
let name = "John";
const age = 25;
console.log(name, age);

Data Types

JavaScript has dynamic typing, with these main data types:

  • Primitive: number, string, boolean, null, undefined.
  • Object: arrays, functions, objects.

Operators

JavaScript supports arithmetic, comparison, logical, and assignment operators.


// Example
let result = 5 + 10; // Arithmetic
if (result > 10) {   // Comparison
    console.log(true && false); // Logical
}
