Java Basics

Syntax Overview

Java syntax forms the basis of how Java code is written and structured. A typical Java program consists of a class definition, methods, and statements. Here's a basic example:


public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!"); // Output text
    }
}

Data Types

Java provides several data types to handle different kinds of data:

  • Primitive Types: byte, short, int, long, float, double, char, boolean
  • Non-Primitive Types: Strings, Arrays, Classes, Interfaces

// Example of data types
int number = 10;
float pi = 3.14f;
boolean isTrue = true;
char grade = 'A';

Variables

Variables in Java are containers for storing data. A variable must be declared with a data type before use.


// Variable declaration and initialization
int age = 25;
String name = "Java";

Operators

Operators are used to perform operations on variables and values. Java provides various operators:

  • Arithmetic Operators: +, -, *, /, %
  • Relational Operators: ==, !=, >, <, >=, <=
  • Logical Operators: &&, ||, !
  • Bitwise Operators: &, |, ^, ~, <<, >>
  • Assignment Operators: =, +=, -=, *=, /=

// Example of operators
int x = 10, y = 20;
int sum = x + y;   // Arithmetic
boolean result = (x < y);   // Relational

Control Flow Statements

Control flow statements dictate the order in which statements are executed. Common control flow statements include:

  • if-else: Executes a block of code based on a condition.
  • switch: Selects a block of code to execute based on a value.
  • for: Executes a block of code for a set number of iterations.
  • while/do-while: Executes a block of code as long as a condition is true.

// Example of if-else
if (x > y) {
    System.out.println("x is greater");
} else {
    System.out.println("y is greater");
}
