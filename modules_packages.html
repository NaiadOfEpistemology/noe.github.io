Modules and Packages

Importing Modules

Modules are files containing Python code that can be imported and used in other Python programs. Use the `import` keyword to import a module.


# Importing the math module
import math

# Using a function from the math module
result = math.sqrt(16)
print(result)  # Output: 4.0

Standard Library Modules

Python's standard library includes a wide range of modules that provide functions and utilities for various tasks.


# Importing the random module
import random

# Using a function from the random module
number = random.randint(1, 10)
print(number)  # Output: A random number between 1 and 10

# Importing the datetime module
import datetime

# Getting the current date and time
now = datetime.datetime.now()
print(now)  # Output: Current date and time

Creating and Using Packages

Packages are directories containing multiple modules. They allow you to organize your code into manageable sections.


# Directory structure:
# mypackage/
#     __init__.py
#     module1.py
#     module2.py

# __init__.py is usually an empty file that indicates the directory is a package.

# Importing modules from a package
from mypackage import module1, module2

# Using functions from the imported modules
module1.function1()
module2.function2()
