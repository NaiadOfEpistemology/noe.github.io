File Handling

Reading Files

Python provides built-in functions to read from files. The open() function is used to open a file in read mode, and the read() method is used to read the content.


# Reading from a file
with open('example.txt', 'r') as file:
    content = file.read()
    print(content)

Writing Files

The open() function is also used to open a file in write mode, and the write() method is used to write content to the file. If the file does not exist, it will be created.


# Writing to a file
with open('example.txt', 'w') as file:
    file.write('Hello, World!')

Working with CSV Files

Python's csv module provides functionality to read from and write to CSV files.


import csv

# Writing to a CSV file
with open('data.csv', 'w', newline='') as file:
    writer = csv.writer(file)
    writer.writerow(['Name', 'Age', 'City'])
    writer.writerow(['Alice', 25, 'New York'])
    writer.writerow(['Bob', 30, 'San Francisco'])

# Reading from a CSV file
with open('data.csv', 'r') as file:
    reader = csv.reader(file)
    for row in reader:
        print(row)

Working with JSON Files

Python's `json` module allows you to work with JSON data, which is commonly used for data exchange.


import json

# Writing to a JSON file
data = {'name': 'Alice', 'age': 25, 'city': 'New York'}
with open('data.json', 'w') as file:
    json.dump(data, file)

# Reading from a JSON file
with open('data.json', 'r') as file:
    data = json.load(file)
    print(data)
