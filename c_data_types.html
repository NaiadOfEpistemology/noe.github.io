Basic Concepts

Tokens

In C programming, tokens are the smallest elements of a program. They are categorized into:

  • Keywords (e.g., int, return)
  • Identifiers (e.g., variable names)
  • Constants (e.g., 10, 'a')
  • Operators (e.g., +, -)
  • Special symbols (e.g., {, })

Identifiers

Identifiers are names given to various program elements such as variables, functions, arrays, etc. They must begin with a letter (a-z, A-Z) or an underscore (_) followed by letters, digits (0-9), or underscores.

Constants

Constants are fixed values that do not change during the execution of a program. They can be of various types, such as integer constants, floating-point constants, character constants, and string literals.

Variables

Variables are used to store data that can be modified during program execution. Each variable must be declared with a specific data type, which determines the kind of data it can hold.

Data types

Data types specify the type of data that a variable can store. C has several built-in data types:

  • Basic Data Types:
    • int: Integer data type.
    • float: Floating-point data type.
    • double: Double-precision floating-point data type.
    • char: Character data type.
  • Derived Data Types:
    • array: Collection of elements of the same type.
    • pointer: Variable that stores the memory address of another variable.
    • structure: Collection of variables of different types.
    • union: Collection of variables that share the same memory location.
  • Enumeration: User-defined data type consisting of integral constants.
  • Void: Represents an empty data type.

Input and output functions

Input and output functions are used to read data from the keyboard and display data on the screen. Common I/O functions in C are:

  • scanf(): Reads formatted input from the keyboard.
  • printf(): Prints formatted output to the screen.

Qualifiers

Qualifiers are keywords used to modify the properties of variables. Common qualifiers in C are:

  • const: Declares a variable as constant, preventing its value from being modified.
  • volatile: Indicates that a variable's value may be changed by external factors.

Modifiers

Modifiers are used to alter the meaning and size of basic data types. Common modifiers in C are:

  • signed: Indicates that a variable can store both positive and negative values.
  • unsigned: Indicates that a variable can store only positive values.
  • short: Reduces the size of an integer variable.
  • long: Increases the size of an integer variable.

Operators

Operators are symbols used to perform operations on variables and values. They are categorized into:

  • Arithmetic Operators: +, -, *, /, %
  • Relational Operators: ==, !=, >, <, >=, <=
  • Logical Operators: &&, ||, !
  • Bitwise Operators: &, |, ^, ~, <<, >>
  • Assignment Operators: =, +=, -=, *=, /=, %=
  • Increment and Decrement Operators: ++, --
  • Conditional (Ternary) Operator: ?::
