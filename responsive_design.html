Responsive Design

What is Responsive Design?

Responsive design ensures web pages adapt to different screen sizes and devices, offering a consistent user experience on desktops, tablets, and mobile phones.

Viewport Meta Tag

The viewport meta tag is crucial for responsive design, as it controls how content is scaled on mobile devices.

CSS Media Queries

Media queries allow you to apply different styles depending on the screen size or device. They are used for creating responsive layouts.


/* Example Media Query */
@media (max-width: 768px) {
    body {
        background-color: lightgrey;
    }
}

Flexible Grids

Using flexible grids with percentage-based widths ensures that layouts resize dynamically with the screen size.


/* Example Flexible Grid */
.container {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
    gap: 10px;
}

Responsive Images

Make images responsive by using the max-width property and relative units like percentages.


/* Example Responsive Image */
img {
    max-width: 100%;
    height: auto;
}

Flexbox and Grid for Responsive Design

Flexbox and Grid are powerful tools for designing responsive layouts. They can adapt content alignment and spacing based on the available space.


/* Example Responsive Flexbox */
.flex-container {
    display: flex;
    flex-wrap: wrap;
    gap: 10px;
}
.flex-item {
    flex: 1 1 200px;
}
