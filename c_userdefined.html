User Defined Datatypes in C

What are User Defined Datatypes?

User defined datatypes in C allow programmers to define their own data types. These can be particularly useful for creating complex data structures. The main user-defined datatypes in C are:

  • Structure
  • Union
  • Enumeration
  • Typedef

Structure

A structure is a user-defined datatype that allows grouping variables of different types together under a single name. Structures are particularly useful for representing complex data items such as records.


// Example of a structure
#include <stdio.h>

struct Person {
    char name[50];
    int age;
    float salary;
};

int main() {
    struct Person person1;

    // Assigning values to structure members
    strcpy(person1.name, "John");
    person1.age = 30;
    person1.salary = 55000.50;

    // Accessing structure members
    printf("Name: %s\n", person1.name);
    printf("Age: %d\n", person1.age);
    printf("Salary: %.2f\n", person1.salary);

    return 0;
}

Union

A union is a user-defined datatype that allows storing different data types in the same memory location. The memory occupied by a union is equal to the memory occupied by its largest member. Unions are useful when you want to use the same memory space for storing different types of data at different times.


// Example of a union
#include <stdio.h>

union Data {
    int i;
    float f;
    char str[20];
};

int main() {
    union Data data;

    // Assigning values to union members
    data.i = 10;
    printf("i: %d\n", data.i);

    data.f = 220.5;
    printf("f: %.1f\n", data.f);

    strcpy(data.str, "C Programming");
    printf("str: %s\n", data.str);

    return 0;
}

Enum

Enumeration (enum) is a user-defined datatype that consists of a set of named integer constants. Enums are useful for defining variables that can only take a specific set of values, making the code more readable and maintainable.


// Example of an enum
#include <stdio.h>

enum Weekday {
    SUNDAY,
    MONDAY,
    TUESDAY,
    WEDNESDAY,
    THURSDAY,
    FRIDAY,
    SATURDAY
};

int main() {
    enum Weekday today;
    today = WEDNESDAY;

    if (today == WEDNESDAY) {
        printf("It's Wednesday.\n");
    }

    return 0;
}

Typedef

The typedef keyword is used to create new names (aliases) for existing data types. This can make the code more readable and easier to maintain.


// Example of typedef
#include <stdio.h>

typedef unsigned long ulong;

int main() {
    ulong num = 123456789;
    printf("num: %lu\n", num);
    return 0;
}
