Functions

Defining Functions

Defining functions allows you to create reusable blocks of code. Functions are defined using the `def` keyword.


def greet():
    print("Hello, World!")
greet()
# Output: Hello, World!

Parameters and Arguments

Parameters are variables that you define within the function, and arguments are the values you pass to the function when you call it.


  def greet(name):
               print(f"Hello, {name}!")
greet("Alice")
# Output: Hello, Alice!

Return Values

Functions can return values using the return keyword.


 def add(a, b):
      return a + b
     result = add(5, 3)
print(result)
# Output: 8

Lambda Functions

Lambda functions are small anonymous functions defined using the lambda keyword.


    multiply = lambda x, y: x * y
    result = multiply(4, 5)
print(result)
# Output: 20

Scope and Global Variables

Scope refers to the visibility of variables within different parts of the code. Global variables are accessible throughout the entire program, while local variables are only accessible within the function they are defined.


global_var = "I am global"

def foo():
    local_var = "I am local"
    print(global_var)
    print(local_var)

foo()
print(global_var)
# Output:
# I am global
# I am local
# I am global

# Uncommenting the following line will raise an error because local_var is not accessible outside the function:
# print(local_var)
