Pointers and Memory Allocation in C

Pointer

A pointer is a variable that stores the memory address of another variable. Pointers are used to access and manipulate memory locations directly, making them powerful tools in C programming.


// Example of a pointer
#include <stdio.h>

int main() {
    int num = 10;
    int *ptr = #

    printf("Value of num: %d\n", num);
    printf("Address of num: %p\n", &num);
    printf("Value stored in ptr: %p\n", ptr);
    printf("Value pointed to by ptr: %d\n", *ptr);

    return 0;
}

Array of Pointers

An array of pointers is an array where each element is a pointer to a variable of the same type. This is useful for handling arrays of strings or other complex data structures.


// Example of an array of pointers
#include <stdio.h>

int main() {
    int num1 = 10, num2 = 20, num3 = 30;
    int *arr[3] = {&num1, &num2, &num3};

    for (int i = 0; i < 3; i++) {
        printf("Value of num%d: %d\n", i + 1, *arr[i]);
    }

    return 0;
}

Pointer to Pointer

A pointer to a pointer is a form of multiple indirection or a chain of pointers. It is a pointer that stores the address of another pointer.


// Example of a pointer to a pointer
#include <stdio.h>

int main() {
    int num = 10;
    int *ptr = #
    int **pptr = &ptr;

    printf("Value of num: %d\n", num);
    printf("Value pointed to by ptr: %d\n", *ptr);
    printf("Value pointed to by pptr: %d\n", **pptr);

    return 0;
}

Pointer Arithmetic

Pointer arithmetic involves performing operations on pointers, such as addition, subtraction, and comparison. These operations are useful for iterating through arrays and other data structures.


// Example of pointer arithmetic
#include <stdio.h>

int main() {
    int arr[5] = {10, 20, 30, 40, 50};
    int *ptr = arr;

    for (int i = 0; i < 5; i++) {
        printf("Value at arr[%d]: %d\n", i, *(ptr + i));
    }

    return 0;
}

Memory Allocation

Memory allocation in C is done dynamically using functions like malloc(), calloc(), realloc(), and free(). These functions are used to allocate and deallocate memory at runtime.


// Example of dynamic memory allocation
#include <stdio.h>
#include <stdlib.h>

int main() {
    int *ptr;
    int n, i;

    printf("Enter number of elements: ");
    scanf("%d", &n);

    ptr = (int*)malloc(n * sizeof(int));

    if (ptr == NULL) {
        printf("Memory not allocated.\n");
        return 1;
    }

    printf("Memory successfully allocated.\n");

    for (i = 0; i < n; i++) {
        ptr[i] = i + 1;
    }

    printf("The elements of the array are: ");
    for (i = 0; i < n; i++) {
        printf("%d ", ptr[i]);
    }

    free(ptr);
    printf("\nMemory successfully freed.\n");

    return 0;
}
