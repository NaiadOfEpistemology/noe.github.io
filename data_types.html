Data Types and Variables

Numbers (integers, floats, complex)

Numbers are used to store numeric values. Python supports integers, floating-point numbers, and complex numbers.

  • Integers: Whole numbers (e.g., 1, -5, 42)
  • Floats: Decimal numbers (e.g., 3.14, -0.001)
  • Complex: Numbers with a real and imaginary part (e.g., 3+4j)

int_num = 10
float_num = 3.14
complex_num = 2 + 3j

Strings

Strings are sequences of characters used to represent text.


text = "Hello, World!"
print(text.lower())  # Output: hello, world!
print(text.upper())  # Output: HELLO, WORLD!
print(text.replace("World", "Python"))  # Output: Hello, Python!
print(text.split(", "))  # Output: ['Hello', 'World!']

Booleans

Booleans represent truth values True or False.


is_valid = True
has_error = False
print(is_valid and has_error)  # Output: False
print(is_valid or has_error)  # Output: True

Lists

Lists are ordered collections of items, allowing for duplicates and heterogeneous data.


fruits = ["apple", "banana", "cherry"]
fruits.append("date")  # ['apple', 'banana', 'cherry', 'date']
fruits.remove("banana")  # ['apple', 'cherry', 'date']
print(fruits.pop(1))  # Output: cherry
fruits.sort()  # ['apple', 'date']

Tuples

Tuples are ordered collections of items that are immutable (cannot be changed).


colors = ("red", "green", "blue", "green")
print(colors.count("green"))  # Output: 2
print(colors.index("blue"))  # Output: 2

Dictionaries

Dictionaries are collections of key-value pairs.


person = {"name": "Alice", "age": 25, "city": "New York"}
print(person.get("name"))  # Output: Alice
print(person.keys())  # Output: dict_keys(['name', 'age', 'city'])
print(person.values())  # Output: dict_values(['Alice', 25, 'New York'])
print(person.items())  # Output: dict_items([('name', 'Alice'), ('age', 25), ('city', 'New York')])

Sets

Sets are unordered collections of unique items.


numbers = {1, 2, 3, 4}
numbers.add(5)  # {1, 2, 3, 4, 5}
numbers.remove(3)  # {1, 2, 4, 5}
evens = {2, 4, 6, 8}
print(numbers.union(evens))  # Output: {1, 2, 4, 5, 6, 8}
print(numbers.intersection(evens))  # Output: {2, 4}

Type Conversion

Python allows you to convert values from one type to another using type conversion functions.


int_num = 10
float_num = float(int_num)  # Converts int to float
str_num = str(int_num)  # Converts int to string
list_str = list("Hello")  # Converts string to list
print(float_num)  # Output: 10.0
print(str_num)  # Output: '10'
print(list_str)  # Output: ['H', 'e', 'l', 'l', 'o']
