Control Structures in C

Simple if

The if statement is used to execute a block of code if a specified condition is true.


if (condition) {
    // Code to be executed if condition is true
}

if-else construct

The if-else statement is used to execute one block of code if a condition is true, and another block of code if the condition is false.


if (condition) {
    // Code to be executed if condition is true
} else {
    // Code to be executed if condition is false
}

if-else if-else ladder

The if-else if-else ladder is used to test multiple conditions sequentially.


if (condition1) {
    // Code to be executed if condition1 is true
} else if (condition2) {
    // Code to be executed if condition2 is true
} else {
    // Code to be executed if all conditions are false
}

Nested if

A nested if statement is an if statement inside another if statement. It allows for more complex condition checks.


if (condition1) {
    // Code to be executed if condition1 is true
    if (condition2) {
        // Code to be executed if condition2 is true
    }
}

Switch case

The switch statement is used to execute one block of code among many, based on the value of a variable or expression.


switch (variable) {
    case value1:
        // Code to be executed if variable is equal to value1
        break;
    case value2:
        // Code to be executed if variable is equal to value2
        break;
    // More cases
    default:
        // Code to be executed if variable doesn't match any case
}

While loop

The while loop is used to execute a block of code repeatedly as long as a specified condition is true.


while (condition) {
    // Code to be executed while condition is true
}

For loop

The for loop is used to execute a block of code a specific number of times. It is often used when the number of iterations is known beforehand.


for (initialization; condition; increment/decrement) {
    // Code to be executed for each iteration
}

Do-while loop

The do-while loop is similar to the while loop, but it guarantees that the block of code is executed at least once, as the condition is evaluated after the block of code has been executed.


do {
    // Code to be executed
} while (condition);

Unconditional Statements

Unconditional statements in C are used to control the flow of the program without any conditions. These include:

  • break: Terminates the nearest enclosing loop or switch statement.
  • continue: Skips the remaining code inside the nearest enclosing loop and proceeds with the next iteration of the loop.
  • goto: Transfers control to the labeled statement.
  • return: Exits the current function and optionally returns a value to the calling function.

// Example of break statement
for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break;
    }
    printf("%d\n", i);
}

// Example of continue statement
for (int i = 0; i < 10; i++) {
    if (i == 5) {
        continue;
    }
    printf("%d\n", i);
}

// Example of goto statement
int main() {
    int x = 10;
    if (x == 10) {
        goto label;
    }
    printf("This will be skipped.\n");
    label:
    printf("This will be executed.\n");
    return 0;
}

// Example of return statement
int add(int a, int b) {
    return a + b;
}
