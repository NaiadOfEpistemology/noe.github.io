File Handling in C

File

A file is a named location that stores data or information permanently. C provides a number of functions to perform basic file operations, including creating, reading, writing, and closing files.

Streams

In C, a stream is a sequence of bytes that acts as an interface between a program and an input/output device, such as a file. Streams can be divided into input streams (for reading data) and output streams (for writing data).

Text and Binary Files

Files in C can be classified into two types: text files and binary files.

  • Text Files: Text files contain data that can be read in a human-readable format. Each line of text is terminated with a newline character (\n).
  • Binary Files: Binary files contain data in a format that is not human-readable. They store data in the same format as it is stored in memory.

File I/O Functions

C provides several functions to handle file input and output operations:

  • fopen(): Opens a file and returns a file pointer.
  • fclose(): Closes a file.
  • fgetc(): Reads a character from a file.
  • fputc(): Writes a character to a file.
  • fgets(): Reads a string from a file.
  • fputs(): Writes a string to a file.
  • fread(): Reads a block of data from a file.
  • fwrite(): Writes a block of data to a file.
  • fprintf(): Writes formatted data to a file.
  • fscanf(): Reads formatted data from a file.

// Example of file I/O
#include <stdio.h>

int main() {
    FILE *file;
    char str[100];

    // Opening a file for writing
    file = fopen("example.txt", "w");
    if (file == NULL) {
        printf("Error opening file.\n");
        return 1;
    }

    // Writing to the file
    fputs("Hello, World!", file);

    // Closing the file
    fclose(file);

    // Opening the file for reading
    file = fopen("example.txt", "r");
    if (file == NULL) {
        printf("Error opening file.\n");
        return 1;
    }

    // Reading from the file
    fgets(str, 100, file);
    printf("Content of the file: %s\n", str);

    // Closing the file
    fclose(file);

    return 0;
}

Formatted I/O Functions

Formatted I/O functions are used to read and write formatted data. These functions include:

  • fprintf(): Writes formatted data to a file.
  • fscanf(): Reads formatted data from a file.

// Example of formatted I/O
#include <stdio.h>

int main() {
    FILE *file;
    int num1, num2;

    // Opening a file for writing
    file = fopen("numbers.txt", "w");
    if (file == NULL) {
        printf("Error opening file.\n");
        return 1;
    }

    // Writing formatted data to the file
    fprintf(file, "%d %d\n", 10, 20);

    // Closing the file
    fclose(file);

    // Opening the file for reading
    file = fopen("numbers.txt", "r");
    if (file == NULL) {
        printf("Error opening file.\n");
        return 1;
    }

    // Reading formatted data from the file
    fscanf(file, "%d %d", &num1, &num2);
    printf("Numbers read from the file: %d, %d\n", num1, num2);

    // Closing the file
    fclose(file);

    return 0;
}

Character I/O Functions

Character I/O functions are used to read and write individual characters. These functions include:

  • fgetc(): Reads a character from a file.
  • fputc(): Writes a character to a file.

// Example of character I/O
#include <stdio.h>

int main() {
    FILE *file;
    char ch;

    // Opening a file for writing
    file = fopen("chars.txt", "w");
    if (file == NULL) {
        printf("Error opening file.\n");
        return 1;
    }

    // Writing characters to the file
    fputc('A', file);
    fputc('B', file);
    fputc('C', file);

    // Closing the file
    fclose(file);

    // Opening the file for reading
    file = fopen("chars.txt", "r");
    if (file == NULL) {
        printf("Error opening file.\n");
        return 1;
    }

    // Reading characters from the file
    while ((ch = fgetc(file)) != EOF) {
        printf("%c", ch);
    }
    printf("\n");

    // Closing the file
    fclose(file);

    return 0;
}
