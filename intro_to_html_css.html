Introduction to HTML & CSS

What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It uses tags to define elements such as headings, paragraphs, links, and images.






    My First Page


    
Hello, World!

    
This is my first web page.

What is CSS?

CSS (Cascading Style Sheets) is a language used to style and layout HTML elements. With CSS, you can control colors, fonts, spacing, and positioning of elements on a page.


/* Example CSS */
body {
    background-color: lightblue;
    font-family: Arial, sans-serif;
}
h1 {
    color: darkblue;
}
p {
    color: black;
}

Uses and Importance

HTML and CSS work together to create visually appealing and functional web pages:

  • HTML: Defines the structure and content of the webpage.
  • CSS: Enhances the design, layout, and responsiveness of the webpage.
  • Both are fundamental skills for web developers and designers.
