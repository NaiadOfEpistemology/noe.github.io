Functions in C

Types of Functions

Functions in C can be categorized into two types:

  • Library Functions: These are pre-defined functions provided by C libraries, such as printf(), scanf(), sqrt(), etc.
  • User-defined Functions: These are functions defined by the programmer to perform specific tasks.

// Example of a user-defined function
#include <stdio.h>

void greet() {
    printf("Hello, World!\n");
}

int main() {
    greet();  // Function call
    return 0;
}

Recursion

Recursion is a technique where a function calls itself. It is used to solve problems that can be divided into smaller, similar problems.


// Example of a recursive function to calculate factorial
#include <stdio.h>

int factorial(int n) {
    if (n == 0) {
        return 1;
    } else {
        return n * factorial(n - 1);
    }
}

int main() {
    int num = 5;
    printf("Factorial of %d is %d\n", num, factorial(num));
    return 0;
}

Call by Value, Call by Reference

Functions in C can be called in two ways:

  • Call by Value: A copy of the actual parameter's value is passed to the function. Changes made to the parameter inside the function do not affect the original value.
    • 
// Example of call by value
#include <stdio.h>

void add(int a, int b) {
    a += b;
    printf("Inside function: a = %d\n", a);
}

int main() {
    int x = 10, y = 5;
    add(x, y);
    printf("Outside function: x = %d\n", x);
    return 0;
}
  • Call by Reference: The address of the actual parameter is passed to the function. Changes made to the parameter inside the function affect the original value.
    • 
// Example of call by reference
#include <stdio.h>

void add(int *a, int *b) {
    *a += *b;
    printf("Inside function: a = %d\n", *a);
}

int main() {
    int x = 10, y = 5;
    add(&x, &y);
    printf("Outside function: x = %d\n", x);
    return 0;
}
