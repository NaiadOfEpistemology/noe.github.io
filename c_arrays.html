Arrays in C

Introduction to Arrays

An array is a collection of elements of the same type stored in contiguous memory locations. It allows you to store multiple values of the same type in a single variable, making it easier to manage and manipulate data.

Arrays are useful when you need to store a large number of related values, such as a list of numbers, characters, or other data types. They can be used to implement various data structures, such as lists, stacks, and queues.


// Declaring and initializing an array
int numbers[5] = {1, 2, 3, 4, 5};

// Accessing array elements
printf("%d", numbers[0]);  // Output: 1

Types of Arrays

There are several types of arrays in C:

  • Single-dimensional Arrays: A single-dimensional array, also known as a one-dimensional array, is a simple list of elements. Each element is accessed using a single index.
    • 
// Declaring and initializing a single-dimensional array
char letters[4] = {'a', 'b', 'c', 'd'};

// Accessing elements in the array
printf("%c", letters[2]);  // Output: c
  • Multi-dimensional Arrays: Multi-dimensional arrays are arrays of arrays. The most common multi-dimensional array is the two-dimensional array, which can be visualized as a table or matrix.
    • 
// Declaring and initializing a two-dimensional array
int matrix[3][3] = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

// Accessing elements in the array
printf("%d", matrix[1][1]);  // Output: 5
  • Three-dimensional Arrays: Three-dimensional arrays are arrays of two-dimensional arrays and can be visualized as a 3D grid.
    • 
// Declaring and initializing a three-dimensional array
int threeD[2][2][2] = {
    {
        {1, 2},
        {3, 4}
    },
    {
        {5, 6},
        {7, 8}
    }
};

// Accessing elements in the array
printf("%d", threeD[0][1][1]);  // Output: 4
